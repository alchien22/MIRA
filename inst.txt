1. Create VM: 
    No GPU: gcloud compute instances create my-vm --machine-type=n1-standard-4 --image-family=ubuntu-2204-lts --image-project=ubuntu-os-cloud --boot-disk-size=100GB --scopes=https://www.googleapis.com/auth/cloud-platform  --zone=us-central1-b
    T4 GPU: gcloud compute instances create my-gpu-vm --machine-type=n1-standard-8 --accelerator=type=nvidia-tesla-t4,count=1 --image-family=ubuntu-2204-lts --image-project=ubuntu-os-cloud --boot-disk-size=100GB --scopes=https://www.googleapis.com/auth/cloud-platform  --zone=us-central1-b

2. Install:
    sudo apt update && sudo apt install -y python3 python3-pip git
    sudo apt update && sudo apt install -y python3.10-venv

3. Stop VM: gcloud compute instances stop my-vm --zone=us-central1-b
4. Verify Stopped: gcloud compute instances list
    Check if STATUS = TERMINATED
5. Resume VM: gcloud compute instances start my-vm --zone=us-central1-b
6. Delete VM: gcloud compute instances delete my-vm --zone=us-central1-b


Check Instances: gcloud compute instances list
Delete Instances: gcloud compute instances delete vm-name
Zones list: gcloud compute zones list

nproc  # Number of CPU cores
free -h  # Available memory (RAM)
df -h  # Disk space usage
